## 2. 项目背景

### 2.1 业务背景

当前通商 e 链业务中，母票签发人有时为核心企业子公司，但兑付时只会查询兑付企业关联的项目中维护的还款账户。由于子公司签发的场景无法识别还款账户，导致兑付失败。

为解决此问题，需要在项目详情参数详情页新增新增通商E链业务网银兑付权限配置功能，允许项目维护人员维护核心企业的子公司信息，以便在兑付时能够正确匹配还款账户。

### 2.2 项目目标

1. 在项目详情参数详情页新增新增通商E链业务网银兑付权限配置
2. 支持新增、删除子公司信息
3. 通过客户名称选择组件选择子公司
4. 确保项目类型字段永远不允许修改

### 2.3 用户价值

| 用户角色 | 价值点 |
| -------- | ------ |
| 项目维护人员 | 可以维护核心企业的子公司信息，提高兑付成功率 |
| 财务人员 | 兑付时能够正确匹配还款账户，减少兑付失败 |
| 系统管理员 | 通过权限控制，确保数据安全 |

## 4. 业务流程

### 4.1 业务流程图

#### 4.1.1 新增流程

```
开始
  ↓
点击新增按钮
  ↓
打开新增弹窗
  ↓
默认客户类型：公司客户
  ↓
点击客户名称输入框
  ↓
打开客户选择弹窗
  ↓
选择客户
  ↓
返回客户 ID 和名称
  ↓
自动填充到表单
  ↓
数据验证
  ↓
├─ 验证通过 → 提交新增请求 → 新增成功 → 刷新列表 → 关闭弹窗 → 结束
└─ 验证失败 → 显示错误提示 → 结束
```

#### 4.1.2 删除流程

```
开始
  ↓
选择要删除的记录
  ↓
点击删除按钮
  ↓
弹出确认对话框
  ↓
├─ 确认删除 → 调用删除接口 → 删除成功 → 刷新列表 → 结束
└─ 取消删除 → 结束
```

### 4.2 异常流程

| 异常 ID | 异常场景 | 处理方式 | 优先级 |
| ------- | -------- | -------- | ------ |
| AE-001 | 接口调用失败 | 显示错误提示，不刷新列表 | P1 |
| AE-002 | 网络异常 | 提示用户检查网络连接 | P1 |
| AE-003 | 数据验证失败 | 显示具体错误信息 | P0 |
| AE-004 | 重复添加客户 | 提示该客户已存在 | P1 |
