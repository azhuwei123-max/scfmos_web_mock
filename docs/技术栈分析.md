## 构建与开发命令

```bash
pnpm i                  # 安装依赖
pnpm dev                # 启动本地开发服务器 (local-dev 环境)
pnpm dev-server         # 启动开发服务器 (dev 环境)
pnpm build:dev          # 构建 dev 环境
pnpm build:test         # 构建 test 环境
pnpm build:stage        # 构建 stage 环境
pnpm build:prod         # 构建 prod 环境
pnpm build:local-dev    # 构建 local-dev 环境
pnpm preview            # 预览构建结果
pnpm ts:check           # TypeScript 类型检查
pnpm lint:eslint        # ESLint 修复
pnpm lint:format        # Prettier 格式化
pnpm lint:style         # Stylelint 修复
pnpm clean              # 删除 node_modules
pnpm clean:cache        # 删除缓存
```

## 项目特点

1. **现代化技术栈**：Vue 3 + TypeScript + Vite 5，性能优秀
2. **低代码能力**：集成 FormCreate 动态表单设计器，支持低表单配置
3. **工作流引擎**：集成 BPMN.js 流程设计器，适合业务流程管理
4. **完善的代码规范**：ESLint + Prettier + Stylelint + Commitlint 全链路检查
5. **国际化支持**：内置 zh-CN / en 双语
6. **按需加载**：AutoImport + Components 自动导入 + Element Plus 按需加载
7. **多环境构建**：支持 local-dev / dev / test / stage / prod 多环境
8. **丰富的组件库**：自定义组件 60+，覆盖业务各种场景
9. **SVG 图标系统**：Iconify + PurgeIcons 按需使用图标
10. **移动端适配**：postcss-pxtorem 实现 REM 适配

## 目录结构概览

```
src/
├── api/              # API 接口层
├── assets/           # 静态资源
├── components/       # 公共组件
├── composables/      # Composition API 复用逻辑
├── compositions/     # 旧版 Compositions（可能逐步迁移）
├── config/           # 配置文件
├── directives/       # 自定义指令
├── hooks/            # 组合式函数
├── layout/           # 布局组件
├── locales/          # 国际化语言包
├── plugins/          # 插件配置
├── router/           # 路由配置
├── store/            # Pinia 状态管理
├── styles/           # 全局样式
├── types/            # TypeScript 类型定义
├── utils/            # 工具函数
└── views/            # 页面组件
